# 🚀 CryptoTracker

A modern cryptocurrency tracking web application that allows users to search, explore, and monitor real-time cryptocurrency market data.


## 🌟 Features

- 🔍 Search cryptocurrencies instantly
- 💰 Real-time price tracking
- 📈 Market statistics and trends
- 🪙 Detailed coin information
- 📱 Fully responsive design
- ⚡ Fast and optimized performance
- 🎨 Modern and clean UI

## 🛠️ Tech Stack

- React.js
- TypeScript
- Tailwind CSS
- REST APIs
- WebSockets
